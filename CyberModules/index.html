List of Learning Modules

These Sociotechnical Cybersecurity Learning modules can be embedded in a project-based course, such as capstone experiences in computing disciplines. The modules may be used as-is or adapted as necessary to fit the course.

M1: Threats and Adversaries
M2: Basic Risk Assessment
M3: Vulnerability Management
M4: Secure Software Development Lifecycle
M5: Cryptography and Public Key Infrastructure (PKI)
M6: Data Security
M7: Intrusion Detection
M8: User Experience and Usability
M9: Ethics
M10: Compliance
